##  Getting Started

### Prerequisites
Make sure to install
- [Python 3](https://www.python.org/downloads/)

### Installation

1. Clone the repo
```sh
git clone https://github.com/gpt-website-builder.git
```
2. Install NPM packages
```sh
pip install -r requirements.txt
```
4. Add your keys
```sh
mv .env.sample .env 
```
3. Run
```sh
python main.py
```
